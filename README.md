# BottomNavigationWithFAB
이 프로젝트는 중간에 FloatingActionButton이 있는 BottomNavigationView를 만드는 방법을 배웁니다.

## Material Design Dependency

프로젝트에 적용된 디펜던시
```
implementation 'com.google.android.material:material:1.6.0-alpha03'
```
